# Bug Report: Grade File Calculation

**Title:** Incorrect grade calculation when the file contains empty cells  
**Description:** System crashes or miscalculates grades when a cell in the input file is empty.  
**Steps to Reproduce:**  
1. Open grades.txt  
2. Leave one cell empty  
3. Run the grade calculation program  
**Expected Result:** Program handles empty cell and calculates correctly  
**Actual Result:** Program crashes or ignores some entries  
**Severity:** Medium
